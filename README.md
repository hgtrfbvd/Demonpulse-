# 🔥 DEMONPULSE SYNDICATE V7

Professional betting intelligence dashboard for Australian greyhound, horse, and harness racing.

## Deploy on Render

1. Push this repo to GitHub
2. Go to [render.com](https://render.com) → New Web Service
3. Connect your GitHub repo
4. Set environment variables:
   - `CLAUDE_API_KEY` = your Anthropic API key
5. Deploy — done.

## Local Development

```bash
pip install -r requirements.txt
python app.py
```

## Tech Stack
- Python / Flask backend
- SQLite database (auto-created)
- Dark terminal UI — red accent theme
- Chart.js for analytics
- Claude API (V7 intelligence engine)
- Gunicorn for production

## Features
- Live race execution panel
- Race simulation engine
- Backtest & AI learning
- Performance analytics
- Bankroll management
- Built-in chat with V7 AI
- Market movement tracking
- Full session logging
